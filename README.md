## 🚀 Hızlı Başlangıç

### Yerel Çalıştırma

```bash
cd yankus-web
node server.js
# http://localhost:3000
```

### Ücretsiz Hosting

#### 1. Vercel (Önerilen)
1. https://vercel.com → GitHub ile giriş
2. "Add New Project" → yankus-web seç
3. Root Directory: public
4. Deploy!

#### 2. Netlify
1. https://netlify.com → "Add new site"
2. public klasörünü sürükle bırak

#### 3. GitHub Pages
1. Settings → Pages
2. Source: main branch, /public folder
